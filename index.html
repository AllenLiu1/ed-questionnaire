seantis-questionnaire by rmt

Seantis Questionnaire is a django questionnaire app which is easily customised and includes advanced dependency support using boolean expressions.

Dependencies

pyparsing

Install

1. Check it out. 2. cd seantis-questionnaire/example 3. ./manage.py syncdb 4. ./manage.py runserver Then point your webserver at http://localhost:8080/ Default admin username is admin with password admin123.
